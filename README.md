# Md2ansi

Md2ansi renders md files to ansi.

## Installation

Add this line to your application's Gemfile:

    gem 'md2ansi'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install md2ansi

## Usage

``` bash
md2ansi foo.md
```
